# NDE WhatsApp AI Product Assistant

AI assistant for ndestore.com that automatically responds to customer inquiries on WhatsApp.

## Features

- Detects vehicle make, model, year and part
- Handles typos and synonyms
- Generates Shopify product search links
- Mobile friendly responses
- Docker deployable
- Railway compatible

## Example

Customer message

Honda Civic 2018 spark plug

AI reply

Product: Spark Plug  
Vehicle: Honda Civic 2018

View options:
https://www.ndestore.com/search?q=spark+plug+honda+civic
